# 附錄 C　名相對照

正文裡，凡遇一個名相，都在它頭一次出現時順手用白話解過了；梵文、巴利文的拼音與變音符號，則一概收在這裡，免得擾了正文。

每一條，給出：**中文名相 — 羅馬拼音（巴/梵）— 一句白話**。標「巴」的是巴利文（早期阿含脈絡），標「梵」的是梵文（論藏脈絡）；有些詞兩者都給，有些是漢譯自有之詞，不另注拼音。

---

## 一、自在與漏盡

- **漏** — āsava（巴）/āsrava（梵）—— 滲漏；煩惱之總名。三漏：欲漏、有漏、無明漏。
- **漏盡** — （漢譯詞）—— 三漏永盡；阿羅漢之所成，全卷之印。
- **自在** — （漢譯詞）—— 本卷取其為「無一物可繫縛」之負面自由（非神力·能耐之積累）。漏盡讀得出的，就是這份自在。
- **阿羅漢** — arahant（巴）/arhat（梵）—— 第四果聖者，漏盡之人；字源義之一即「應供」（見下）。
- **自證定型句** — （漢譯詞）—— 「我生已盡，梵行已立，所作已作，自知不受後有」；漏盡者自知自證之語，全卷反覆出現之結文。

## 二、第二部——心解脫・慧解脫

- **心解脫** — cetovimutti（巴）—— 心於貪等煩惱得解脫、無所繫縛之相。
- **慧解脫**（解脫義） — paññāvimutti（巴）—— 以慧斷漏而得之解脫。（與下「慧解脫阿羅漢」一詞所指之人相應。）
- **不動心解脫** — akuppā cetovimutti（巴）—— 心解脫之最勝相：無退動（無因緣可退）、不待時（入定不待緣）。解脫到了底的名字。
- **不時解脫** — （論藏譯詞）—— 不待時節因緣即得之解脫；與「時解脫」相對，即不動心解脫之異名。
- **時解脫** — （論藏譯詞）—— 須待勝緣方入定、須恒護恐退之解脫；前五種性阿羅漢之相，本卷取以對照不動。
- **六種阿羅漢** — （論藏分判）—— 退法·思法·護法·安住法·堪達法·不動法；前五從信解（鈍根）生為時解脫，第六從見至（利根）生為不時解脫。本卷僅取「可退／不退」一對照，不入部派退義之諍。
- **慧解脫**（聖者） — paññāvimutta（巴）—— 不得滅盡定、但以慧斷煩惱障而漏盡的阿羅漢。非「解脫不全」，乃「以慧斷漏已全」。
- **俱解脫** — ubhatobhāgavimutta（巴）—— 兼得滅盡定（八解脫身作證）與漏盡的阿羅漢；慧定二力，兼離煩惱障與解脫障。
- **煩惱障** — kleśa-āvaraṇa（梵）—— 貪瞋癡諸漏，繫人於生死之障；二類阿羅漢同斷此障（漏盡齊一）。
- **解脫障**（定障） — vimokṣa-āvaraṇa（梵）—— 障最深定（八解脫·滅盡定）現起之障；唯俱解脫者兼離之。
- **八解脫** — aṭṭha vimokkhā（巴）—— 由「色觀色」起、歷四無色定、終於第八「想受滅」的八重定境；其第八即滅盡定。
- **名珠寶** — （漢譯詞·喻）—— 轉輪王七寶中最珍之「珠」；經以之喻不動心解脫於諸成就中之最勝地位。

## 三、第三部——三明六通

- **三明** — tisso vijjā（巴）—— 宿命明、天眼（生死）明、漏盡明；六通中之第五·第二·第六於無學位之攝。
- **六神通**（六通） — chaḷabhiññā（巴）—— 神足（如意足）、天眼、天耳、他心、宿命、漏盡六種；前五依四靜慮、異生亦得，唯第六漏盡通唯聖所得。
- **漏盡通** — āsavakkhaya-abhiññā（巴）—— 六通之第六，漏盡之認知面（漏盡之自知）；唯聖者得，是諸通之樞。
- **三際愚** — （論藏譯詞）—— 對前際（過去）、後際（未來）、中際（現在當下）之愚；三明如次對治之，唯漏盡明治中際愚——當下正繫縛的這一念。
- **異生** — pṛthagjana（梵）—— 凡夫；未見道、未斷見惑者。前五通異生亦得。
- **三示現** — （漢譯詞）—— 神足變化示現、他心示現、教誡示現；佛化導之三式，唯教誡（說法）令人漏盡解脫。
- **智慧眼** — （漢譯詞）—— 照見無我、能自照斷漏之眼；阿那律陀抑「天眼」而標此為無上。

## 四、第四部——滅盡定

- **滅盡定**（想受滅） — nirodha-samāpatti（梵）/saññāvedayitanirodha（巴，想受滅）—— 八解脫之第八、九次第定之頂；身口意三行俱滅、想受俱滅，而壽暖不離、諸根不壞——止息之極而非斷滅。
- **九次第定** — navānupubbavihārā（巴）—— 四靜慮·四無色·滅盡定，依次而入的九級定境；滅盡定為其殿。
- **無想定** — asaññā-samāpatti（梵）—— 僅滅「想」而想知不全滅之定；與滅盡定之「想知俱滅」異，止息較淺。
- **壽・暖・識** — āyu, usmā, viññāṇa（巴）—— 維繫生命之三法；捨身而死則三者俱捨，入滅盡定則壽暖不離（識之微辨見正文）。
- **現法涅槃** — diṭṭhadhamma-nibbāna（巴）—— 於現生即可證入之涅槃境；聖者以此勝解入滅盡定而無怖。
- **起滅自在** — （漢譯詞）—— 於滅盡定之入、出、起、滅得自主堪能；成俱解脫之判準在此，非在「漏盡更深」。
- **寂滅樂** — （漢譯詞）—— 以一切受之止息為樂；想受滅之「至樂」即此，非受之樂。四樂之一（離欲樂·遠離樂·寂滅樂·菩提樂）。

## 五、第五部——不受後有・應供

- **不受後有** — nāparaṃ itthattāya（巴，意「無此後有」）—— 此生命終後不再受後世生死相續；了結的是「受後有之苦」，非論斷「死後之我」有無。
- **後有** — punabbhava（巴）—— 後世之生存相續。
- **無記** — avyākata（梵）—— 置而不答之問；非因不知（「世尊非不知、非不見」），乃因問題錯置。
- **十四無記** — （漢譯詞）—— 世間常無常四句·世有邊無邊四句·命身一異二句·如來死後有無四句，共十四；皆預設一固定實體而問其屬性，故捨置。
- **四記** — （論藏分判）—— 答問之四式：一向記（直答）、分別記（分別而答）、反詰記（反問而答）、捨置記（置而不答）；十四無記屬捨置記。
- **應供** — arahant（巴）/arhat（梵）—— 「阿羅漢」一名之字源義：諸有染者所應供；自在之社會面，世間讀解脫之德而生之尊。
- **福田** — puññakkhetta（巴）—— 世間植福之田；阿羅漢以解脫之德為「無上福田」。其福非阿羅漢求取，乃世間緣其德而生——無所求故田淨。
- **無學** — asekha（巴）—— 無可再學者；阿羅漢之位。福田人分學人（十八）、無學人（九）。
- **五分法身** — （漢譯詞）—— 戒、定、慧、解脫、解脫知見五者所成之德身；聖眾以此「可敬可重，可奉可供，世良福田」。
- **無諍** — araṇā（梵）—— 阿羅漢自知福田中勝，恐他人緣己起煩惱，故修之護他之智；無所求貫徹於福田身分之證。
- **偷婆**（塔） — thūpa（巴）/stūpa（梵）—— 供奉舍利之塔；四種應起偷婆者：轉輪聖王、漏盡阿羅漢、辟支佛、如來。
- **轉輪聖王** — cakkavatti（巴）/cakravartin（梵）—— 以正法治世之理想君王；經中與如來、漏盡阿羅漢並列為「世人所應供養」之一。
- **辟支佛** — paccekabuddha（巴）/pratyekabuddha（梵）—— 無師自悟、不立教化之覺者；四種應起偷婆之一。
