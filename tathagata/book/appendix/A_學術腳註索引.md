## 第一部 · 名滿之地

**1.1 十個名，一個人**（依 TATHAGATA-P01）
- 經：《八十華嚴》卷五十〈如來出現品〉，如來以無量法而得出現、非以一緣非以一事（普賢菩薩告如來性起妙德等大眾，T10n0279_p0263a19）。《大般涅槃經》（北本）卷一〈壽命品〉，佛臨涅槃於拘尸那國娑羅雙樹間，文殊師利舉東方虛空等如來「十號具足」為一佛標準全名（T12n0374_p0370c15）。
- 論：《大智度論》卷五十六，佛自追述昔受然燈佛授記、十號一氣連屬整列（T25n1509_p0457b06）；卷二，逐號釋名——如來（T25n1509_p0071b16）、應供、正遍知（同段）、明行足之三明滿不滿（T25n1509_p0071c13）、善逝至天人師（T25n1509_p0072a29）、佛與世尊（T25n1509_p0073a02）；卷二，名滿即在滿之樞——「獨佛應當受佛名號」「佛二法滿足故稱勝一切人」（T25n1509_p0073a23）。
- 誌異：其一，著者譬「父·師·醫·鄰」（同一人於四種關係被喚出四名，喻十號為一自在之十面），正文已標為比方。其二，P01 標《智論》卷五十六該段為〈釋顧視品第三十〉；naive head-finder 取前品〈散華品〉，品名經 Code 覆核確認（XML 卷五十六〈顧視品第三十〉；書體無品號，不受影響）。

**1.2 兩份十自在**（依 TATHAGATA-P02）
- 經：《八十華嚴》卷三十八〈十地品·第八不動地〉（場景在他化自在天王宮，金剛藏菩薩承佛神力說十地），八地菩薩十自在——命、心、財、業、生、願、解、如意、智、法（T10n0279_p0200b14）；八地菩薩得寂滅忍、本可入究竟涅槃而諸佛起其智門（T10n0279_p0199b01）；十方諸佛親勸「我等十力、無畏、十八不共諸佛之法，汝今未得」（同段 p0199b01）。卷四十六〈佛不思議法品〉（世尊神力加持青蓮華藏菩薩，由其答），「諸佛世尊有十種自在法」——與八地十自在分品別立（T10n0279_p0245a06）。
- 論：《十地經論》卷十〈不動地〉，八地之智自在唯能「示現」如來力、無畏、不共、相好（T26n1522_p0183c21）。《大智度論》卷二，唯佛全具十號——「獨佛應當受佛名號」（回扣 1.1，T25n1509_p0073a23）。
- 誌異：其一，八地十自在與諸佛十種自在法兩套十目之一一對應，經無明文；本書明言「不勉強去配，留著」（屬判攝會通，作者保留）。其二，八地十自在三譯本（八十華嚴／玄奘《十地經》T0287／世親《十地經論》T1522）用字次第互異，P02 並陳；書體不入正文。其三，被破之誤喻「調亮一輪／放大一輪」（喻「同一自在之明暗」讀法），章內以反詰標明非經義。

## 第二部 · 三身

**2.1 現出身來**（依 TATHAGATA-P03）
- 經：《解深密經·如來成所作事品》，曼殊室利（文殊）菩薩請問佛「如來法身有何等相」，佛答「於諸地波羅蜜多善修出離、轉依成滿，是名如來法身之相」；又問答二乘唯解脫身、法身唯佛（本書經由《瑜伽師地論》卷七十八〈攝決擇分〉之引文取證，T30n1579_p0733c16）。
- 論：《成唯識論》卷十〈究竟位〉，三身全列——自性身、受用身（自受用·他受用）、變化身（T31n1585_p0057c20）；三身由轉依所顯、四法（智）分攝三身（T31n1585_p0058a06）。《攝大乘論本》卷下〈彼果智分〉，報化二身「依」法身、自性身為一切法自在轉所依止（T31n1594_p0149a18）。《佛地經論》卷七，三身三種常——法身本性常、報身不斷常、化身相續常（引《莊嚴論》，T26n1530_p0326b01）。
- 誌異：其一，著者譬「真容（對至交）·禮容（對賓客）·俯就之容（對童蒙）」（一人依所對者而現之三容，喻三身依機而現），正文已標。其二，「三身常住」係唯識／莊嚴論一系之判；三身（尤報化）常無常、法身是否色法，諸宗論藏有廣辯，本書標屬唯識系判，不裁諸宗高下。其三，《佛地經論》之卷次，本書 dump 之 juan-milestone 對 p0326 顯偏移，實為卷七（依 P03 已物理覆核）。

## 第三部 · 四智

**3.1 轉，不是得**（依 TATHAGATA-P04，全章論藏）
- 論：《成唯識論》卷十〈究竟位〉，四智全列與定義——大圓鏡智、平等性智、妙觀察智、成所作智（T31n1585_p0056a12）；四智是「轉」八、七、六、五識而得，智強識劣、為勸依智捨識故說轉識得（T31n1585_p0056b02）；四智攝三身（標明係「有義」一說，論另存他義，T31n1585_p0058a08）。《佛地經論》卷三，平等性智「建立佛地無住涅槃」（繫末部 P10，T26n1530_p0302a19）；卷一，佛地五法＝清淨法界＋四智（T26n1530_p0291b15）。《攝大乘論本》卷下，四智自在由轉識蘊依（T31n1594_p0149c06）。
- 誌異：其一，著者譬「磨去蒙垢之鏡」（垢磨盡則朗照自現，非垢去後另搬一新鏡，喻四智為轉染成淨所顯、轉非得）；按「大圓鏡現眾色像」之鏡象本屬經論，「磨垢」之引申為著者譬，正文已標。其二，四智攝三身之配當係《成唯識論》「有義」一說，論另存他義並陳；本書取一說、標其為論中一說，不代裁跨論定論。其三，「轉識成智／轉識得智」係方便表詮（為勸依智捨識），本書明標智非識、亦非離識別有，勿執實有一智從識變生。

## 第四部 · 十力四無畏十八不共

**4.1 如實知，師子吼**（依 TATHAGATA-P05）
- 經：《八十華嚴》卷十七〈梵行品〉，正念天子問法慧菩薩「云何得梵行清淨、從菩薩位逮無上菩提」，法慧菩薩答，列「如來十力」十智為應觀察修習者；第八、九徑作「天眼無礙智、宿命無礙智」、末作「永斷習氣智」（T10n0279_p0088c21）。
- 論：《大智度論》卷二十四，十力——每力以「如實知，某力也」結，力即智力（T25n1509_p0235c22）；卷二十五，四無畏第一無畏為式「我是一切正智人……乃至不見是微畏相……如師子吼」（T25n1509_p0241b24）；卷二十五，四無畏體、二自利二利他、力與無畏之分（T25n1509_p0242a22）。
- 誌異：其一，承第三部「磨鏡」喻——十力「照於十境而蔽它不得」係此喻之引申，章內已標。其二，十力末「永斷習氣智」所涉「佛漏盡力深於二乘」之分判，係作者保留，本書照錄不裁。

**4.2 無失，無礙**（依 TATHAGATA-P06）
- 經：《八十華嚴》卷十四〈賢首品〉，文殊師利菩薩以偈問賢首菩薩、賢首以偈答；偈舉十力、無畏、十八不共為「如來自在用，悉以示現度眾生」（本部之眼，T10n0279_p0075a06）。
- 論：《大智度論》卷二十六，十八不共法全列——三無失、六無減（遮遣）、三業隨智（相順）、三世智無礙（T25n1509_p0247b11）；「不共」之揀別——前十八中聲聞辟支佛有分、後十八中無分（T25n1509_p0247b19）。卷二十五，四無礙——以「地」為例、通達無滯是名無礙（T25n1509_p0246a22）。
- 誌異：其一，「不共」與菩薩卷「異質而非高下」之會通、二乘菩薩有分無分之精確界定，係作者保留，本書照錄不裁高下。其二，四無礙之因位／佛地分際、地別智別諸論異說，本書照錄不代調和。其三，主調「無礙」三入佛德名相——天眼／宿命無礙智（十力）、三世智無礙（十八不共）、四無礙智。其四，本部所涉《涅槃》卷二十七「佛性者所謂十力四無所畏大悲……一切眾生悉有佛性」（T12n0374_p0525c03）一段，涉佛性普遍性、常樂我淨、一闡提、二乘究竟，皆作者保留，本書未引入。

## 第五部 · 相好大悲一切種智

**5.1 看得見的印**（依 TATHAGATA-P07）
- 經：《八十華嚴》卷四十六〈佛不思議法品〉，青蓮華藏菩薩列諸佛十種最勝無上莊嚴，相好為「第一最勝無上身莊嚴」（T10n0279_p0244b22）。
- 論：《大智度論》卷二十九，「為生身故說三十二相，為法身故說無相」「法身以十力四無所畏四無礙智十八不共法諸功德莊嚴」、相好為度生而現、現相不破無相法（T25n1509_p0274a12）；卷四，相好修因所感——「過三阿僧祇劫然後種三十二相業因緣」（繫燃燈佛授記，T25n1509_p0086c17）。
- 誌異：其一，「印」之結構說法——相好＝色身之印（看得見），對第一部十號＝名言之印（讀得出），二印皆標識無相之自在自體（法身·智德）而非其本質；此係本書結構主喻，非經文，章內以「正像第一部的十號」明繫。其二，《智論》卷次：相好生身法身之判為卷二十九（P07 自校，原檢索作卷三十四）。其三，《涅槃》卷八〈如來性品〉「佛性如是不可思議，三十二相八十種好亦不可思議」（T12n0374_p0410b13）一段，涉佛性、常樂我淨之我、相好繫佛性之了義，皆作者保留；華嚴「第一最勝無上身莊嚴」已足立「相好非無關」，故本書未引此段。

**5.2 悲為根本**（依 TATHAGATA-P08）
- 經：《大般涅槃經》（北本）卷十一〈現病品〉，佛示病相而臥、大眾以偈哀請（舉「三世諸世尊，大悲為根本……若無大悲者，是則不名佛」），世尊大悲熏心、即從臥起、以相好莊嚴其身放大光明（T12n0374_p0429c08）。《八十華嚴》卷五十〈如來出現品〉，普賢菩薩述如來大慈大悲「依大方便善巧、依如來出現、依無礙慧光明」（T10n0279_p0264c20）；卷二十四〈十迴向品〉，金剛幢菩薩承佛神力說頌，菩薩「發心趣入一切種智」（T10n0279_p0128c06）。
- 論：《大智度論》卷二十七，大慈與樂大悲拔苦之界說（T25n1509_p0256b15）、「諸佛心中慈悲名為大」繫十八不共法次第（T25n1509_p0256b22）、一切智（總相·因）／一切種智（別相·果·觀種種法門）之判（T25n1509_p0258c29）；卷十八，「般若波羅蜜在佛心中變名為一切種智」「菩薩求度名波羅蜜、佛已度名一切種智」（T25n1509_p0190a21）。
- 誌異：其一，「同體大悲」一語於本三經字面查無（係後世述語）；本書未用，唯用三經本文等義段——大慈大悲、大悲為根本、諸佛心中慈悲名為大、大慈救護大悲度脫。其二，論文本身之譬（牢獄諸子、入海深淺、前燈後燈）皆《智論》本文，章內已標「這是論裡打的比方」。其三，因位果位名異（般若→一切種智）與「異質而非高下」之會通、一切種智〔果〕與菩薩般若〔因〕之確切異同、二乘有無一切種智，係作者保留，本書照錄不裁；同段「一切智是聲聞辟支佛事」之二乘 line 亦屬此，本書避開不展。其四，《涅槃》此偈 locus：P08 footnote 作〈梵行品〉，XML 實為〈現病品第六〉（T12n0374 卷十一），經 Code 覆核確認（書體無品號，不受影響）。

## 第六部 · 圓滿之自在

**6.1 不費一點力**（依 TATHAGATA-P09）
- 經：《八十華嚴》卷三十八〈十地品·不動地〉，金剛藏菩薩說八地菩薩十自在（因位，T10n0279_p0200b14）；卷四十六〈佛不思議法品〉，青蓮華藏菩薩列「諸佛世尊十種自在法」、末以「一念一切現……而未曾用少方便力」為第十自在法（果位·任運無功用，T10n0279_p0245a06）。《大般涅槃經》（北本）卷二，佛告諸比丘——「世間亦有常樂我淨，出世亦有常樂我淨；世間法者有字無義，出世間者有字有義」、四德對配如來法身／涅槃／佛／正法（T12n0374_p0377b21）；卷七〈如來性品〉，佛答「我者即是如來藏義，一切眾生悉有佛性即是我義」、貧女金藏喻（T12n0374_p0407b09）。
- 誌異：其一（全書最重之保留）：大涅槃「常樂我淨」之「我」德了義——我／無我之會通（經自陳「無我者名為生死、我者名為如來」並列）、涅槃「真我」與凡夫妄我之分判、一切眾生悉有佛性之普遍性（一闡提）、此「我＝佛性」與相好繫佛性（P07）之了義——皆如來藏一系核心判定；本書照錄經文而不代裁，章內明立「一條必須守住的線」，講法留予作者。其二，諸 locus 經 Code 端 CBETA XML 覆核確認（書體無品號，不受影響）：諸佛十自在法 P09 footnote 作〈離世間品〉、XML 實為〈佛不思議法品第三十三〉（卷四十六）；四德段北本品名為〈壽命品第一〉（卷二；南本〈哀歎品〉）；我即如來藏段 XML 實為〈如來性品第四之四〉（卷七；P09 footnote 作之三、本附錄初稿誤作之一，已正）。

**6.2 不住，即自在**（依 TATHAGATA-P10，全書末章·兩軸收卷）
- 經：《仁王般若波羅蜜經》卷上〈菩薩教化品〉，佛說五忍·寂滅忍——「無明盡相為金剛，盡相無相為薩婆若」（T08n0245_p0826c24）；月光王偈「始入金剛一切了……一轉妙覺常湛然」（T08n0245_p0827c23）。
- 論：《成唯識論》卷十，四種涅槃、第四無住處涅槃「真如出所知障、大悲般若常所輔翼、不住生死涅槃、利樂有情窮未來際、用而常寂」、「二乘無學容有前三，唯我世尊可言具四」（T31n1585_p0055b07）；卷十、卷九，金剛喻定一剎那二障頓盡、入如來地（T31n1585_p0054a29、p0053c20）。《大智度論》卷二十一〈念佛〉，「諸佛得無礙解脫故於一切法中智慧無礙」「根本拔煩惱及習……名為無礙解脫」（T25n1509_p0220c20）；卷四十〈五眼〉，菩薩次第入金剛三昧得一切種智、成就十力四無畏四無礙十八不共大慈大悲（收本部諸德於成佛一念，T25n1509_p0350b08）。
- 誌異：其一，妙覺接縫段為本書系兩軸（行軸《等覺妙覺》SAMBODHI-P03 ∥ 證軸本卷）共用同段、兩讀——行軸讀「無明盡相為金剛」（道之最後一步）、證軸讀「盡相無相為薩婆若」（自在更無餘步之究竟）。此「兩讀」之最終裁定（接縫義理），係作者保留，本書只標兩讀之相，不代主張。其二，圓成（如來究竟無住處涅槃·唯佛具四）與漸成（菩薩無住處涅槃·因位進程）之會通、與菩薩卷「異質而非高下」之跨軸會通、二乘無餘與佛無住處之同異、二乘究竟、迴小向大，皆作者保留，本書照錄不裁。其三，「無礙解脫」之傳統梵語還原 apratihata-vimokṣa、無住處涅槃之梵語 apratiṣṭhita-nirvāṇa，皆為入括之梵語註記，非漢譯本文，書體正文不列梵字（見附錄 C）。其四，仁王該段 locus：P10 footnote 作〈受持品·寂滅忍偈〉，XML 實為〈菩薩教化品第三〉（卷上），經 Code 覆核確認。
