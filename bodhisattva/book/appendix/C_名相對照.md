# 附錄 C　名相對照

正文裡，凡遇一個名相，都在它頭一次出現時順手用白話解過了；梵文拼音與變音符號，則一概收在這裡，免得擾了正文。

每一條，給出：**中文名相 — 羅馬拼音（梵）— 一句白話**。本書為大乘分冊，拼音以梵文為主；漢譯自有之詞、或屬後世判教所立、或屬本書綜整之讀，不另注拼音，逕標（漢譯詞）（判教名）（本書讀法）。

---

## 一、兩種自在與其分判

- **自在** — vaśitā／aiśvarya（梵）—— 無礙、自由；本書所讀者，是走這條路所成的那一種自由（到達之後的自由）。
- **菩薩自在** — （本書主題）—— 拒那終極之寂、盡未來際利他的自由；其印記是「能取那證而不取」（能出而不出）。
- **阿羅漢自在** — （本書系雙生卷主題）—— 取那終極之寂、眾苦永寂、不受後有的自由；其印記是「能出而出」。本書系《阿羅漢自在》專說。
- **異質而非高下** — （本書鎖定語）—— 二自在是異構的兩種、各得其真，不是同一階梯上的高下。
- **不取證／不住涅槃** — （漢譯詞）—— 到了該取證、可入寂之處而不取；菩薩自在最深的印記，與聲聞之取證相對。
- **分判** — （本書讀法）—— 判二自在為異構之兩種（如判魚游與鳥飛之異），非判孰高孰低。

## 二、門與根

- **菩提心／發菩提心** — bodhicitta（梵）—— 為度一切眾生而求無上菩提之心；菩薩自在所印之處，一念可入。
- **菩薩** — bodhisattva（梵）—— 發菩提心、行菩薩道之人；自發心即預菩薩數。
- **大乘** — mahāyāna（梵）—— 以成佛度生為目的之乘；與二乘為平行別途。
- **聲聞／辟支佛（獨覺）** — śrāvaka／pratyekabuddha（梵）—— 取證四沙門果之行者／無佛之世自悟之聖者；合稱二乘。
- **種姓** — gotra（梵）—— 堪任成佛之根性；菩薩種姓兼證所知障淨（異於二乘但證煩惱障淨）。
- **一切智／一切種智** — sarvajñatā／sarvākārajñatā（梵）—— 佛之無所不徹之智；發心所求之的。
- **二資糧** — puṇya-jñāna-saṃbhāra（梵）—— 福德與智慧兩種資糧；菩薩自在以「積量」者，無量無際。
- **三阿僧祇劫** — asaṃkhyeya-kalpa（梵，阿僧祇）—— 三段無量數之劫；二資糧積成之長。
- **迴小向大** — （漢譯詞）—— 二乘迴心轉趣大乘之說；其是否、如何，本書不裁，屬作者宗義抉擇。

## 三、六度十度與四攝

- **六波羅蜜／六度** — ṣaṭ-pāramitā（梵）—— 布施、持戒、忍辱、精進、禪定、般若；菩薩自在之形。
- **波羅蜜** — pāramitā（梵）—— 義為「到彼岸」；五度得般若導，方擔得起此名。
- **檀・尸羅・羼提・毘梨耶・禪・般若** — dāna／śīla／kṣānti／vīrya／dhyāna／prajñā（梵）—— 即布施、持戒、忍辱、精進、禪定、智慧。
- **福智二分** — （漢譯詞）—— 六度之經緯：施戒忍屬福分，般若屬智分，精進禪定通二分。
- **十度／後四度** — daśa-pāramitā（梵）—— 六度之外加方便、願、力、智；後四非新功課，是後得智回身入世之起用，扶成前六。
- **方便・願・力・智** — upāya／praṇidhāna／bala／jñāna（梵）—— 後加之四度。
- **無分別智／後得智** — nirvikalpa-jñāna／（漢譯詞）—— 根本智直照真如、不起分別；後得智於其後起，回入差別世間起用。
- **四攝** — catuḥ-saṃgraha-vastu（梵）—— 布施、愛語、利行、同事；自在純就利他所現之形。
- **同事** — （漢譯詞）—— 自己先圓滿所勸之善、與所化事同；自在於關係中可見之樞紐。

## 四、十地與無生法忍

- **十地** — daśabhūmi（梵）—— 菩薩自登地到佛果之十級階位。
- **歡喜・離垢・發光・焰慧・難勝・現前・遠行・不動・善慧・法雲** — pramuditā／vimalā／prabhākarī／arciṣmatī／sudurjayā／abhimukhī／dūraṃgamā／acalā／sādhumatī／dharmameghā（梵）—— 十地逐名。
- **不動地** — acalā（梵）—— 第八地；無能壞、無能動，捨功用而任運。
- **無生法忍** — anutpattika-dharma-kṣānti（梵）—— 信住「諸法本不生、如虛空性」之見；成就此忍即入八地。忍，認可安住之義，非忍耐。
- **現觀** — （漢譯詞）—— 當場親見實相，非隔著概念去推知。
- **任運／無功用** — anābhoga（梵）—— 不假作意用力而自運轉；八地捨功用、得無功用法之相。
- **阿鞞跋致／不退轉** — avaivartika（梵）—— 不退轉；經論明文中「不退」之本名。
- **十自在** — （漢譯詞）—— 八地菩薩所得：命、心、財、業、生、願、解、如意、智、法（玄奘本譯字小異）；對治十種怖畏。係菩薩因位之自在，與「諸佛世尊十種自在法」分屬不同名目。

## 五、二涅槃與二障

- **涅槃四種** — （漢譯詞）—— 本來自性清淨、有餘依、無餘依、無住處；一切有情有初一，二乘無學容有前三，唯佛具四。
- **無餘依涅槃** — nirupadhiśeṣa-nirvāṇa（梵）—— 出煩惱障、煩惱既盡、餘依亦滅、眾苦永寂、不受後有；阿羅漢自在所取之終極之寂。
- **無住處涅槃** — apratiṣṭhita-nirvāṇa（梵）—— 出所知障、大悲般若雙運、不住生死涅槃、盡未來際利他、用而常寂；菩薩自在之究竟核。名相屬唯識／攝論所立。
- **轉依** — āśrayaparāvṛtti（梵）—— 捨離惑而不捨離生死，轉染成淨；菩薩之「廣大轉（為捨不捨）」異於聲聞之「下劣轉（一向背生死）」。
- **煩惱障／所知障** — kleśa-āvaraṇa／jñeya-āvaraṇa（梵）—— 二障各障一果：煩惱障障涅槃（解脫）、所知障障菩提（覺智）；二乘但斷前者，菩薩俱斷。
- **不受後有** — （漢譯詞）—— 不再受未來之生死身；阿羅漢自在之印，與菩薩「盡未來際」相對。
- **大悲般若** — mahākaruṇā／mahāprajñā（梵）—— 大悲故不住涅槃、般若故不住生死；雙運令二邊俱不住。
- **唯佛具四** — （漢譯詞）—— 四涅槃唯佛圓具；無住處涅槃之果位圓滿屬佛，本書取菩薩因位之趣向相，果位圓滿屬如來自在分區。

## 六、其餘

- **法空真如／法無我** — dharma-nairātmya（梵）—— 諸法無實自性之真如；所知障所覆，斷之則顯、大悲般若得發。
- **三十七道品** — （漢譯詞）—— 三十七種趣道之行法；菩薩具足觀知而不取證之所修。
- **留惑潤生** — （漢譯詞）—— 菩薩不盡取證、留一分以資受生度眾之機制。
- **阿耨多羅三藐三菩提** — anuttarā samyak-saṃbodhi（梵）—— 無上正等正覺；發心所求之果。
- **南無本師釋迦牟尼佛** — namo …（梵，皈敬）—— 皈敬本師釋迦牟尼佛之語。
