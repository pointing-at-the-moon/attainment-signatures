# The Attainment Signatures
## 證——自在的簽名:阿羅漢、菩薩、如來

*The fourth gate of the [指月 / Pointing-at-the-Moon](https://github.com/pointing-at-the-moon) fourfold 信解行證 — attainment signed as mastery, read across the three masteries: the Arhat's, the Bodhisattva's, and the Tathāgata's.*

華嚴以**信、解、行、證**四字攝盡修道全程。八部曲立其**信**,六行門明其**解**,行階位走其**行**;此系列收於**證**——不講路,講**到達後的自在**。

> 到達與自在 · attainment signed as mastery.

證不是另一條路,是路走完之後蓋下的簽名。三重自在,自下而上:**阿羅漢自在**(出世之自在)→ **菩薩自在**(二資糧無量之自在)→ **如來自在**(十自在)。śrāvaka → bodhisattva → Buddha。

### 與其他三軸的接縫 / Seams with the other axes

證(本軸)與行(`realization-paths`)是結構上的攣生兄弟,在兩端相接:

- 行軸之 **四沙門果·阿羅漢果** ──► **阿羅漢自在**(兩軸第一分區)
- 行軸之 **等覺妙覺·妙覺** ──► **如來自在**(兩軸末分區)

行講「走到」,證講「到達後的自在」。階位是路,自在是抵達的簽名。
